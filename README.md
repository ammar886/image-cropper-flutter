# image-cropper-flutter
This is a custom widget for image cropper functionality, this has been built primarily to be used as a custom widget in Flutterflow and is based on crop_your_image package on pub.dev

Follow the below steps to implement this in Flutterflow:
- Add the following dependency: _**crop_your_image: ^2.0.0**__
- Create a custom widget
- Do the following inport in your widget: _**import 'package:crop_your_image/crop_your_image.dart';**_
- Use the widget on your UI

**References**

- **crop_your_image 2.0.0**: https://pub.dev/packages/crop_your_image
